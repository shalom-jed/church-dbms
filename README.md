# Church Management System

A comprehensive web-based church management system for managing members, small groups, attendance, departments, events, and finances.

## 🚀 Features

- **Member Management**: Track members and family units
- **Small Groups**: Manage 10+ small groups with weekly attendance
- **Sunday Service Attendance**: Individual and bulk tracking
- **Departments**: Multi-department membership with attendance
- **Events**: Event management with expense tracking
- **Finance**: Income/expense tracking, pledges, reports
- **Reports**: Visual charts, PDF/Excel export, automated email reports
- **Communication**: Email notifications and announcements

## 🛠️ Tech Stack

### Frontend
- React 18 + TypeScript
- Vite
- shadcn/ui + Tailwind CSS
- Recharts
- React Hook Form + Zod

### Backend
- Node.js + Express + TypeScript
- PostgreSQL (Supabase)
- Prisma ORM
- JWT Authentication
- Resend (Email)

### Deployment
- Frontend: Vercel
- Backend: Render
- Database: Supabase
